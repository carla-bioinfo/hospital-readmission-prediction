# 🏥 Hospital Readmission Risk Prediction & Analytics

**Projeto de Data Science para Predição de Readmissão Hospitalar**

## 📊 Sobre

Sistema end-to-end para prever risco de readmissão de pacientes em 30 dias.

## 🎯 Objetivo

Identificar pacientes de alto risco para intervenção proativa.

## 📈 Roadmap
ETAPA 0: Setup ✅

ETAPA 1: EDA ⏳

ETAPA 2: SQL ⏳

ETAPA 3: ML ⏳

ETAPA 4: Dashboard ⏳
## 🚀 Quick Start

```bash
git clone https://github.com/carla-bioinfo/hospital-readmission-prediction.git
cd hospital-readmission-prediction
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
jupyter notebook
```
